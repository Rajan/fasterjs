
## Build Setup

``` bash
# install dependencies
npm install

# start mongodb
mongod --dbpath path_to_database

database folder is focus-gestor-orcamentos/server/database

# start server in one console
node server.js

# serve with hot reload at localhost:8080 (another console)
npm run dev

# some people might have errors with modules not being properly installed, even after running `npm install`
run `npm install -only=dev`

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```
